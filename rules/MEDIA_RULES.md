## Media-запросы (обязательно к соблюдению)

1. Все media-запросы пишутся ТОЛЬКО:
   - в конце файла
   - после всех базовых стилей

2. Запрещено:
   - вставлять media-запросы между стилями
   - дублировать одинаковые media-запросы внутри одного файла
   - разбивать один блок по разным media-запросам

3. Логика:
   Сначала все базовые стили → потом один сгруппированный блок @media
   Основные стили должны располагаться до медиазапросов!

4. Один брейк = один блок @media:
   ❌ Плохо:
   @media(...) { .a {} }
   ...код...
   @media(...) { .b {} }

   ✅ Правильно:
   @media(...) {
     .a {}
     .b {}
     .c {}
   }

Используемые брейкпоинты:
- 768px
- 1280px
- 1400px
