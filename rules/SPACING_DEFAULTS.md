# SPACING_DEFAULTS.md
**Разрешённые базовые отступы (v1)**

## Назначение
Этот документ определяет **ЕДИНСТВЕННЫЕ отступы**, которые Cursor имеет право
использовать без отдельного разрешения пользователя.

Любые другие отступы, размеры и интервалы считаются **дизайном**
и **запрещены без явной команды**.

---

## 1. Отступы секций (РАЗРЕШЕНО)

```css
.section {
  padding-block: 48px;
}

@media (min-width: 1280px) {
  .section {
    padding-block: 72px;
  }
}
```

**Правила:**
- Разрешено использовать `.section` без запроса
- Запрещено:
  - менять значения
  - добавлять модификаторы
  - задавать другие отступы секций

---

## 2. Отступы между абзацами (РАЗРЕШЕНО)

```css
p {
  margin: 0;
}

p + p {
  margin-top: 12px;
}
```

**Правила:**
- Только для обычного текста
- Запрещено:
  - использовать `margin-bottom`
  - применять к карточкам, спискам, формам

---

## 3. Заголовки (ОГРАНИЧЕНО)

```css
h2, h3, h4 {
  margin: 0;
}
```

**Правила:**
- Заголовки не имеют внешних отступов по умолчанию
- Расстояние до контента:
  - либо за счёт `.section`
  - либо по явной команде пользователя

---

## 4. Grid-gap (РАЗРЕШЕНО, системное)

```css
.row {
  gap: var(--col-gap-mob);
}

@media (min-width: 1280px) {
  .row {
    gap: var(--col-gap-desk);
  }
}
```

**Правила:**
- Grid-gap — часть инфраструктуры
- Запрещено:
  - менять значения
  - задавать gap внутри карточек

---

## 5. Что СТРОГО ЗАПРЕЩЕНО без команды пользователя

- padding внутри карточек
- margin между элементами внутри секции
- gap между кнопками
- spacing в формах
- любые декоративные отступы
- любые состояния (hover / focus / active)

---

## 6. Поведение при сомнениях

Если отступ:
- не описан в этом документе
- не указан пользователем
- неочевиден по макету

→ **остановись и задай вопрос**
→ **ничего не добавляй по умолчанию**

---

## Краткая формула

> Если отступ не описан в SPACING_DEFAULTS.md —
