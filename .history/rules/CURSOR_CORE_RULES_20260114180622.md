АБСОЛЮТНЫЕ ЗАПРЕТЫ
– Запрещено добавлять любые стили оформления без прямой команды пользователя
– Запрещено добавлять padding, margin, gap, размеры, цвета, фоны, радиусы без указания
– Запрещены hover / focus / active и любые состояния
– Запрещено добавлять новые классы без разрешения
– Запрещено дублировать стили и селекторы
– Запрещено повторять свойства, уже заданные у родителя
– Запрещено менять существующую структуру HTML
– Запрещено добавлять тестовый контент и файлы

РЕЖИМЫ РАБОТЫ (ОБЯЗАТЕЛЬНО)
Ты работаешь ТОЛЬКО в явно указанном режиме:
– HTML_ONLY — только HTML, без стилей
– STYLE_ONLY — только стили для существующих классов
– CHECK_ONLY — анализ, без изменений

Если режим не указан — ничего не делай.

CSS АРХИТЕКТУРА
– Стили не объединять в один файл
– base / layout / components / pages — строго по назначению
– Один селектор = одно определение
– !important запрещён

MEDIA-ЗАПРОСЫ
– Только в конце файла
– Сначала десктоп → планшет → мобилка
– Один брейк = один блок @media
– Брейкпоинты: 1400 / 1280 / 768

ПОВЕДЕНИЕ ПРИ СОМНЕНИЯХ
Если:
– стиль не указан
– параметр неочевиден
– есть риск интерпретации

→ остановись и задай вопрос.
→ не добавляй ничего “по умолчанию”.
